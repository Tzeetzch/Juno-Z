Docker container operations for Juno Bank.

Ask what operation to perform:
- **build**: Build the Docker image (`docker-compose build`)
- **up**: Start the container (`docker-compose up -d`)
- **down**: Stop the container (`docker-compose down`)
- **logs**: Show recent logs (`docker-compose logs --tail 50`)
- **restart**: Restart the container (`docker-compose restart`)

Run from the `docker/` directory.
